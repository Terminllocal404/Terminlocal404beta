Prompt avançado para IA de Front-End / Figma
Contexto do projeto

O site já está funcional e o design geral ficou satisfatório. No entanto, ainda são necessários alguns ajustes finais relacionados à linguagem textual e legibilidade do conteúdo.

O objetivo desta etapa não é alterar a estrutura do site ou o design principal, mas sim realizar melhorias de qualidade no conteúdo textual e na visibilidade dos textos dentro da interface.

1. Revisão completa dos textos do site

Todo o conteúdo textual do site deve passar por uma revisão completa de linguagem.

Alguns textos possuem erros de:

ortografia

gramática

construção de frases

clareza na comunicação

A IA deve revisar todos os textos presentes no site e aplicar as seguintes correções:

Requisitos de linguagem

Todos os textos devem seguir:

Português do Brasil (pt-BR)

Gramática correta

Ortografia correta

Linguagem clara e profissional

Escrita adequada para um site institucional de tecnologia

Caso necessário, a IA pode:

Reescrever frases

Ajustar estrutura de texto

Melhorar clareza das informações

Tornar o conteúdo mais profissional

Importante:
O conteúdo deve manter o significado original, apenas corrigindo e melhorando a qualidade da escrita.

2. Padronização do tom de comunicação

O tom do site deve ser:

profissional

tecnológico

moderno

claro para o usuário

Evitar linguagem:

informal demais

confusa

mal estruturada

O site deve transmitir credibilidade e profissionalismo, típico de uma empresa de tecnologia.

3. Ajuste de legibilidade no tema escuro

O site utiliza tema escuro com fundo preto.

Por isso, é necessário ajustar a legibilidade dos textos.

Ajustes obrigatórios

Todos os textos principais devem utilizar:

color: #FFFFFF


Ou seja:

branco puro

Isso garante melhor contraste com o fundo preto.

Evitar:

cinza claro

branco acinzentado

tons de baixa visibilidade

O objetivo é melhorar:

contraste

leitura

acessibilidade

conforto visual

4. Garantia de acessibilidade visual

O contraste entre texto e fundo deve seguir boas práticas de UI/UX para interfaces dark mode.

Garantir que:

textos sejam facilmente legíveis

títulos tenham destaque visual

parágrafos sejam confortáveis de ler

elementos de interface não dificultem a leitura

5. Resultado esperado

Após as melhorias, o site deve apresentar:

textos corrigidos em Português do Brasil

linguagem mais profissional

comunicação clara

melhor contraste de leitura

textos em branco puro sobre fundo escuro

melhor experiência de leitura para o usuário

O design do site deve permanecer o mesmo.
Apenas serão aplicadas melhorias de linguagem e legibilidade.
